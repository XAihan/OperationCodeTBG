# actionCode

这是模仿行动代号桌游写的简易版，直接按照下面的命令启动就可以。
开始游戏之后分为队长入口和队员入口。
队长入口可以看到所有牌的颜色。
其中，红蓝为各自队伍内容。
灰色代表切换答题权。
黑色标识爆炸，选中黑色的队伍直接在本次游戏中失败。
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
